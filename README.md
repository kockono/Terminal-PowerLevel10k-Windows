### 1. Descargar Font 
https://www.nerdfonts.com/font-downloads
### 2. Instalar Wsl 
Abrir Terminal `PowerShell`
```sh
wsl --install
```
### 3. Selecionar distro y version 2
```sh
wsl --set-version Ubuntu 2
```
